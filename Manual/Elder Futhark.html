On the Subject of Elder Futhark

"Fehu, Uruz, Teiwaz, Hagalaz, Ansuz, Raido, Kenaz"

Elder Futhark consists of 23 different runes. A sequence of 6 runes is present on the module. Your goal is to press the correct runes in the correct sequence.

In order to do so you need to identify the name of the runes (See Appendix RAIDO: Runes and their names)

After that you need to generate a keyword for every rune (See Appendix KENAZ: Determining a keyword)

Lastly encrypt the name of the runes using the Vigenère Cipher (See Appendix WUNJO: Vigenère Ciper)

The currently active rune is white. The defuser needs to input the encrypted name of the active rune using the 23 runestones on the module (See Appendix TEIWAZ: Runes and their corresponding letters)

If the defuser clicked on a wrong runestone the active rune resets. In order to solve the module every rune must be passed. A rune is passed when it`s encrypted name was successfully entered using the runestones.

Appendix KENAZ: Determining a keyword

In order to determine a keyword you have to follow the list top to bottom for the rune you want the keyword to create. You need to create a letter from the number you get in every step. A=0, B=1, C=2 and so on.

Always apply everything where the conditions are fullfilled. Always drop all decimals in every calculation you make.

  • If the rune contains an E or an O the letter is (last digit of the serial number * 10) % 26
  • If the rune is in an even position the letter is (number of modules on the bomb / number of letters in the serial number * number of port plates) % 26
  • If the rune is in the third or sixth position the letter is number of Off Indicators ! % 26
  • If the length of name of the rune is less than 5 the letter is (number of indicators + number of battery holders) % 26
  • The letter is the position of the rune - 1
  • The letter is (length of the name of the rune * 10) % 26
  • The letter is (position of the rune + length of the name of the rune) % 26

Appendix WUNJO: Vigenère Ciper

The Vigenère Cipher works with a keyword. Example: Plaintext = futhark, keyword = ice. Now you need to repeat the keyword till it matches in length with the plaintext: iceicei

Then you look in the top row of the table for the first letter of the plaintext and in the left column for the first letter of the keyword. The matching letter is the first letter of our encrypted word

Using this method for the whole word means that futhark with keyword ice equals nwxpcvs in Vigenère Cipher.
